Steps to open and run NB.py
1. Extract tar content in the same folder
2. Open NB.py using Microsoft Visual Studio.
3. Make sure to have the most popular python extension installed. Can use the extension tab in the left to search for it. (Ctrl+Shift+X)
4. Open the explorer tab on the left (Ctrl+Shift+E), select open folder.
5. Select the folder containing the NB.py file (should be the default when clicking select folder)
6. Make sure that processed-movies-test.txt and processed-movies-training.txt have processed vectors. They should already be processed by pre-process.py by default.
7. Click the run icon for NB.py. Results should be show up as the final-output.txt file and final-parameters.txt file. The results are already there by default.
